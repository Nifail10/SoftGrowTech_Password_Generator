# SoftGrowTech Web Development Internship - Task 2
## Project 2: Password Generator Pro 🔐

### 📌 Overview
[cite_start]This project is an advanced, interactive Password Generator built during my internship at **SoftGrowTech**[cite: 1, 32]. [cite_start]It focuses on providing a secure way for users to generate random, high-entropy passwords while maintaining a "simple and clean interface"[cite: 38].

### 🚀 Advanced Features
- **Live Generation:** The password updates instantly as you adjust the length slider or toggle settings—no extra clicks needed!
- **Dark Mode Support:** A sleek theme switcher to toggle between Light and Dark modes for better accessibility.
- **Dynamic Strength Meter:** A real-time visual indicator that calculates password security based on length and character variety.
- [cite_start]**Customizable Criteria:** Options to include Uppercase letters, Numbers, and Symbols to meet specific security requirements[cite: 35].
- **One-Click Copy:** Integrated clipboard functionality with visual feedback when a password is copied.
- **Intro Animations:** Smooth CSS-based entrance animations for a modern, professional feel.

### 🛠️ Technologies Used
- [cite_start]**HTML5:** Semantic structure for the generator interface[cite: 31].
- [cite_start]**CSS3:** Custom properties (variables) for Dark Mode, Flexbox for layout, and Keyframes for animations[cite: 31].
- [cite_start]**JavaScript (ES6):** Logic for randomization, character mapping, and DOM manipulation[cite: 37].

### 📂 Project Setup
1.  [cite_start]**Repository Name:** `SoftGrowTech_Password_Generator`[cite: 17, 18].
2.  **Files:**
    - `index.html`: Contains the structure and theme toggle.
    - `style.css`: Contains the responsive design and Dark Mode logic.
    - `script.js`: Contains the generation algorithm and event listeners.

### 📖 How to Run
- Clone the repository.
- Open `index.html` in any modern browser.
- Adjust the slider and checkboxes to see the live password generation in action!
